Q. [평균 구하기](http://codingdojang.com/scode/610?answer_mode=hide)
리스트에 있는 숫자들의 평균을 구하는 프로그램을 만들어라.

    [4, 6, 8] = 6
    [11, 17, 20, 24] = 18
    [26, 33, 45, 51, 60] = 43

    평균 : 자료의 값의 총합을 자료의 개수로 나눈 값

A.
'''python
class printMeanValue:
    def __init__(self, list):
        self.list = list

    def printMV(self):
        length = len(self.list)
        sum = sum(self.list)
        return int(sum/length)
'''
